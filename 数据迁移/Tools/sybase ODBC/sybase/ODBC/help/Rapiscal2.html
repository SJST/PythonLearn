API Functions

All Sybase ASE ODBC Drivers are ODBC Level 1–compliant—they support all ODBC Core and Level 1 functions. They also support a limited set of Level 2 functions. The drivers support the functions listed in Table B-1 and Table B-2. Any additions to these supported functions or differences in the support of specific functions are listed in the "ODBC Conformance Level" section in the individual driver topics.
