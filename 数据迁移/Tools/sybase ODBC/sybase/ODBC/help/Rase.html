The Sybase Wire Protocol Driver

The Sybase ASE ODBC driver (or "Sybase Wire Protocol driver") supports the following Sybase database servers:

·         Sybase Adaptive Server 11.5 and higher

·         Sybase Adaptive Server Enterprise 12.0, 12.5, and 12.5.1

The Sybase Wire Protocol driver is supported in the Windows environments.

See the README file shipped with your Sybase product for the file name of the Sybase Wire Protocol driver.
